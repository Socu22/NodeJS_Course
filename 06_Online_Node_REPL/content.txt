How do you set up a website & run it ? 
For us developers to run node we use the node REPL, with the command "node fileName.js" at the correct path, or through ../folder/fileName.js.
You can also choose to download a tool "npm install nodemon", as it is global you can use it wherever you want, it is the same syntax, "nodemon fileName.js". 
Nodejs uses 'npm' as a build tool that you can use to install dependencies in a package.json file for your websites in a package/folder.  
To setup use the command "npm init" then "npm install express", or you could set up an package.json yourself & then
tell it to install its dependencies through "npm install" when setting up local websites. 
{
    "dependencies": {
        "express" : "5.2.1"
    }
}
These dependencies are downloaded into node_modules together with other automatically configured data to get nodejs to run on this operating system, 
that package acts as a library for the website. Each of the other fileName.js is also known as a module in nodejs.

how a rest api endpoints look like ?
GET /beers → Retrieve all beers
GET /beers/{id} → Retrieve a specific beer by ID
POST /beers → Create a new beer (data in body)
PUT /beers/{id} → Fully update/replace a beer by ID
PATCH /beers/{id} → Partially update a beer by ID
DELETE /beers → Delete a beer (based on body)
